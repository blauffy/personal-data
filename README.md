# PII Anonymization Pipeline для русскоязычных текстов

[![Python 3.10+](https://img.shields.io/badge/python-3.10+-blue.svg)](https://www.python.org/downloads/)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**Автоматическое распознавание и обезличивание персональных данных (ПДн) в документах на русском языке.**  
Проект выполнен в рамках курсовой работы по дисциплине «Цифровые технологии защиты информации» (МЭИ, 2025).  
Гибридный подход: регулярные выражения для структурированных форматов + эвристики для ФИО.  
Обеспечивает идемпотентность замен и сохранение семантики текста.

## Основные возможности

- Распознавание **10 типов** персональных данных в свободном тексте  
- Обезличивание с сохранением формата (маскирование / синонимическая замена)  
- Идемпотентность — одинаковые исходные данные заменяются одинаково  
- Генерация синтетического тестового корпуса (150 текстов, каждый тип ПДн ≥50 экземпляров)  
- Оценка качества распознавания (F1‑мера, seqeval) и семантики (BERTScore)  
- Модульная структура, готовая к интеграции в реальные системы

## Типы распознаваемых ПДн

| Категория | Атрибуты |
|-----------|----------|
| **Документы** | ИНН, паспорт РФ, СНИЛС, номер свидетельства о браке, кадастровый номер |
| **Финансы** | Номер банковской карты |
| **Контакты** | Номер телефона (РФ), URL‑адрес |
| **ФИО** | Имя, фамилия (с определением рода) |


## Установка и запуск

### 1. Клонировать репозиторий
```bash
git clone https://github.com/ваш_логин/personal-data.git
cd personal-data 
```
### 2. Создать виртуальное окружение
```bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS/Linux
source .venv/bin/activate
```
### 3. Установить зависимости
```bash
pip install -r requirements.txt
```
### 4. Запустить пайплайн
```bash
python main.py
```
## Результаты качества

| Метрика | Значение |
|---------|----------|
| **Weighted F1-score (распознавание)** | 0.934 |
| **BERTScore (сохранение семантики)** | 0.82 |
Детальный отчёт по классам выводится в консоль.
## Используемые технологии

-  **Python 3.10+** — основной язык реализации  
- `razdel` — токенизация русского текста (без зависимостей)  
- `bert-score` — оценка семантической близости исходного и обезличенного текста  
- `seqeval` — метрики для NER (BIO-разметка)  
- `torch` — бэкенд для вычислений BERTScore
